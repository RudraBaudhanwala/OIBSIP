# Data Cleaning Project

## Objective

This project demonstrates professional data cleaning techniques using Python and Pandas.

## Dataset

Titanic Dataset

## Tasks Performed

* Data Quality Report
* Missing Value Handling
* Duplicate Removal
* Data Standardization
* Outlier Detection
* Data Type Correction
* Before vs After Summary

## Tools Used

* Python
* Pandas
* NumPy
* Jupyter Notebook

## Output

A cleaned dataset ready for analysis was generated and saved as CSV.
