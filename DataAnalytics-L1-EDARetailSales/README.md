# 📊 Task 1: Exploratory Data Analysis (EDA) on Retail Sales Data

## 📌 Objective

The objective of this project is to perform Exploratory Data Analysis (EDA) on a Retail Sales dataset to discover meaningful patterns, trends, and customer purchasing behavior using Python.

## 🛠️ Tools & Libraries Used

* Python
* Jupyter Notebook
* Pandas
* NumPy
* Matplotlib
* Seaborn

## 📂 Dataset

* **Dataset Name:** Retail Sales Data
* **Format:** CSV

## 📈 Analysis Performed

* Data Loading and Inspection
* Data Cleaning and Preparation
* Summary Statistics
* Monthly Sales Analysis
* Gender-wise Sales Analysis
* Age Group Analysis
* Product Category Analysis
* State-wise Sales Analysis
* Occupation-wise Sales Analysis
* Marital Status Analysis
* Data Visualization using Bar Charts and Count Plots

## 🔍 Key Insights

* Adult age group customers made the highest number of purchases.
* Female customers contributed more to total sales than male customers.
* Clothing and Food categories recorded higher sales.
* Sales varied across different states and occupations.
* Monthly sales trends helped identify peak sales periods.
